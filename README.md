# Portfólio Profissional - Ruan

Este repositório contém um site de portfólio profissional completo, incluindo seções de apresentação, currículo, contato e muito mais.

## Características do Projeto

- **Full-Stack**: Aplicação com frontend em React e backend em Express
- **Design Responsivo**: Adaptado para desktop, tablet e dispositivos móveis
- **Multi-idioma**: Suporte para diferentes idiomas
- **Seção de Currículo**: Exibição detalhada de experiências profissionais
- **Formulário de Contato**: Funcional para receber mensagens

## Tecnologias Utilizadas

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Estilização**: Tailwind CSS, shadcn/ui
- **Hospedagem**: Render

## Estrutura do Projeto

```
├── client/           # Código do frontend React
├── server/           # Código do backend Express
├── shared/           # Schemas e tipos compartilhados
├── public/           # Arquivos estáticos
├── render.yaml       # Configuração para deploy no Render
```

## Desenvolvimento Local

Para executar o projeto localmente:

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`
4. Acesse `http://localhost:5000` no navegador
